## 2️⃣ Escolha seu papel (clique no seu)

### 🔍 **SDR — Prospecção**
- Seu dashboard mostra: leads pendentes, follow-ups vencidos
- Pipeline: você vê apenas "PROSPECÇÃO"
- Ação rápida: clique "WhatsApp" no lead → conversa direta
- Próximo passo: registre contato + agende follow-up

### 🤝 **CLOSER — Negociação**
- Seu dashboard mostra: carteira de negociações em andamento
- Pipeline: você vê apenas "NEGOCIAÇÃO"
- Ação rápida: clique "Ligar" do lead → chamada direta
- Próximo passo: altere status conforme avanço (Reunião → Proposta → Fechado)

### 📊 **GESTOR — Gestão**
- Seu dashboard mostra: visão geral (todos os pipelines)
- Menu: Equipe (team), Resultados (KPIs), Relatórios (gerar)
- Ação rápida: vá para "Resultados" → veja performance de cada SDR/closer
- Próximo passo: gere relatório semanal (menu "Relatórios")

### ⚖️ **ADVOGADO — Processos**
- Seu dashboard mostra: "Movimentações Recentes" (últimas 8)
- Cada linha = uma ação judicial atualizada pelo CNJ
- Badge "🔴 Nova" = movimento do dia (urgente!)
- Próximo passo: click no processo → veja detalhes completos

### 📞 **RECEPCÃO — Atendimento**
- Seu dashboard mostra: visão geral (consultas)
- Acesso: Dashboard + Acompanhamento
- Ação: receba chamada de cliente → consulte status em "Movimentações Recentes"
- Próximo passo: registre atividade no lead

## 3️⃣ Seu Primeiro Contato (SDR/Closer)

### 🚀 Em 3 cliques:

```
1. Pipeline (sidebar) → escolha seu lead
2. Card do lead → clique "WhatsApp" (verde) ou "Lilar" (azul)
3. Converse! O link abre direto no seu celular
```

### 🚀 Em 4 cliques (registrar):

```
1. Clique no card do lead → abre Drawer (lado direito)
2. Scroll down → "Registrar contato agora" (botão azul)
3. Escolha data do próximo follow-up → "Agendar"
4. Pronto! Ele aparece no seu "Follow-ups" amanhã
```

## 5️⃣ Features Rápidas por Papel

### SDR
```
💡 Copiar template de mensagem?
   Menu "Templates" → copiar → colar no WhatsApp

🔥 Filtrar leads por prioridade?
   Pipeline → dropdown "Prioridade" → escolha ALTA/MEDIA/BAIXA

📊 Ver sua performance?
   Menu "Resultados" → veja sua linha na tabela
```

### CLOSER
```
💌 Enviar e-mail de proposta?
   Menu "Templates" → tab "E-mail" → copiar proposta formal

📅 Agendar reunião?
   Card → Drawer → date picker → "Agendar"

🎯 Seu fechamento na semana?
   Dashboard → card "Taxa de conversão" = seu resultado
```

### GESTOR
```
📈 Performance do time?
   Menu "Resultados" → tabela "Performance por responsável"

📋 Relatório semanal?
   Menu "Relatórios" → selecione "Semanal SDR" ou "Semanal Closer" → "Gerar"

⚡ Atribuir múltiplos leads?
   Menu "Ações em Massa" → "Atribuir leads" → escolha responsável
```

### ADVOGADO
```
🚨 Novo movimento urgente?
   Dashboard → "Movimentações Recentes" → badge "🔴 Nova"

📍 Consultar status de processo?
   Dashboard → clique na linha → abre detalhes do processo

⏰ Prazos próximos?
   Movimentações → procure por "audiência" ou "intimação"
```

## 🆘 Erro? Teste isto

### "Não consigo ligar"
✓ Verifique se tem telefone preenchido (Pipeline → card)  
✓ Seu navegador permite tel: links? (teste com outro lead)  
✓ Tente "WhatsApp" em vez de "Lilar"

### "Follow-up não aparece"
✓ Você clicou "Agendar"? (não basta escolher data)  
✓ Verificou a data? (hoje não conta)  
✓ Vá para Dashboard → "Follow-ups" → procure

### "Não vejo movimentações novas"
✓ DataJud roda 1x/dia (09:00 UTC)  
✓ Verifique amanhã de manhã  
✓ Ou vá em Automações → rode manual (em breve)

### "Erro na página"
✓ Limpe cache (CTRL+SHIFT+DEL)  
✓ Saia e entre de novo  
✓ Se persistir → reporte ao TI
